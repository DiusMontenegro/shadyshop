# Shady Shop (MERN) E-commerce App

## Technologies Used

- Shady UI
- ReactJS
- Redux
- Bootstrap
- Express
- Node.js
- MongoDB
- Postman

## Features

### Basic

- Search product
- Add to cart
- Delete from cart
- Edit order
- Checkout through PayPal
- Place an order
- Register
- Sign in

### Admin

- Edit user name & email
- Delete user
- Set user as admin
- Add product
- Remove product
- Mark as delivered
- Edit product
